## ðŸš€ Publishing Steps

1. **Update version** (if needed):
   ```bash
   npm version patch  # 1.0.0 -> 1.0.1
   npm version minor  # 1.0.0 -> 1.1.0
   npm version major  # 1.0.0 -> 2.0.0
   ```

2. **Verify package contents**:
   ```bash
   npm pack --dry-run
   ```

3. **Login to npm**:
   ```bash
   npm login
   ```

4. **Publish**:
   ```bash
   npm publish
   ```

5. **Verify on npm**:
   - Check: https://www.npmjs.com/package/detect-in-app-browser
   - Test installation: `npm install detect-in-app-browser`

6. **Create git tag**:
   ```bash
   git tag v1.0.0
   git push origin v1.0.0
   ```
